Introduction

This application is currently for demonstration purposes. It is part of a project undertaken to produce a source of autogenerateable antibiotic associated quizes for healthcare students to practice. This project is lead by PharmD students from the Class of 2022 at the University of Pittsburgh School of Pharmacy. and is developed by Ahmed Bilal

Developed By: Ahmed Bilal

Instructions

The questions, their blanks and their answers are randomly generated depending on the drug classes selected by the user. If the user does not select any class then only a single class is used, Glycylcyclines. Everything is generated when the user clicks the button "Generate Options", and the "Show Answer" button will show the correct answer. Everytime the "Generate OPtions" button is clicked a new set of question, blank and answers will be generated depending on the selected drug classes.
